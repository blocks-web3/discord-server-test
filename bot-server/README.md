# bot-server

## Settings

### .env

#### DISCORD_TOKEN

Please create your bot and get Token.  
https://discordjs.guide/preparations/setting-up-a-bot-application.html

#### OPENAI_API_KEY

Please create your account.  
https://platform.openai.com/account/api-keys

## Install

```
pnpm install
```

## Run Server

```
pnpm start
```
